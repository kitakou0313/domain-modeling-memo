# 型の理解
ドメインのモデリングを関数型の手法で実装

- 用語の定義
    - 関数
        - 入力と出力を持つブラックボックス
    - 型
        - 関数の入力または出力として使用可能な値の集合に与えられた名称
        - 関数も値として利用できる
            - int -> (int -> int)
                - int型の値を受け取り、(int -> int)という型シグネチャの関数を返す という関数の型シグネチャ
- F#での関数
    - 型シグネチャという記法で表される
        - 例:
            - let add1 x = x + 1
                - int -> int
            - let add2 x y = x + y
                - int -> int -> int
        - 各パラメータは矢印で区切られる
    - 定義
        - letキーワードを使って定義
        - 返り値は明示的に書かなくていい
        - 関数の中にサブ関数を定義できる
        - ジェネリック型を利用できる
            - F#では型の名前に'がつく
    - 値の定義と型の定義が似ている
    - ケース名
        - 型そのものではなく、その型に属する別のケース名
        - 選択型の場合どちらもその型
- 型の合成
    - 複数の型を合成して異なる型を作成する
    - 合成の種類
        - AND型（直積型）
            - 内部に複数の型を持つオブジェクト
        - OR型（直和型）
            - 複数の型のうちどれか一つを用いる
            - F#では判別共用体
        - 単純型
            - 既存の型を別の型として定義する
        - 代数的な型システム
            - 全ての複合型がより小さな型をANDまたはORで合成してできているもの
